Knockout+AMD+DesignPatterns

Esta é uma simples aplicação exemplo de como utilizar classe base padrão para derivar todos os 'Models' do Knockout. Pois possui alta flexibilidade para realização de testes unitários de seus componentes: ViewModels, Models e Controllers.

O conceito da utilização de encapsulamento à utilização de componentes de terceiros permite a utilização de mock para prover dados para teste.
